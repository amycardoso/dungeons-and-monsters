# Dungeons & Monsters

A dungeon crawler game built with React and TypeScript. Navigate through 3 levels, collect chests, dodge enemies, and escape!

  <img src="https://raw.githubusercontent.com/amycardoso/dungeons-and-monsters/master/gamegif.gif">
</p>

## How to Play

- **Arrow keys** to move (or D-pad on mobile)
- Collect all chests, then reach the exit door
- Avoid enemies and traps — you have 3 hearts
- Pick up power-ups: heart, shield, speed boost
- Complete all 3 levels to win

## Levels

| Level | Name | Enemies | Chests |
|-------|------|---------|--------|
| 1 | The Entrance | 1 demon | 2 |
| 2 | The Dungeon | 2 demons, 2 mini-demons | 4 |
| 3 | The Abyss | 2 demons, 3 mini-demons | 5 |

## Running Locally

```bash
cd dmgame
npm install
npm run dev
```

Open http://localhost:3000

## Tech Stack

- React 18
- TypeScript 5
- Vite 6
